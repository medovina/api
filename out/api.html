  • Request

Query Parameters

  • Valid: (boolean)

    Get only valid or invalid instances

  • Language: (string - minLength: 2 - maxLength: 2)

    Get only instances with given language

post /instances

Create a new instance for a separate institution

  • Request

Query Parameters

  • Instance Name: (string)

    Name of the instance

    Example:

    MFF UK, Prague
  • Owner: (string)

    Owner of the instance

    Example:

    John Doe
  • Language: (string - minLength: 2 - maxLength: 2)

    Default app language of the instance specified by the two letter standard ISO 639-1

    Example:

    en
