/instances

/instances get post

get

Get the list of all instances

post

Create a new instance for a separate institution

get /instances

Get the list of all instances

Secured by oauth_2_0

ReCodEx supports OAuth 2.0 for authenticating API requests.

  • Request
  • Security

Query Parameters

  • Valid: (boolean)

    Get only valid or invalid instances

  • Language: (string - minLength: 2 - maxLength: 2)

    Get only instances with given language

Secured by oauth_2_0

Headers

  • Authorization: (string - pattern: ^Bearer: .+$)

    Used to send a valid OAuth 2 access token. Do not use with the "access_token" query string parameter.

HTTP status code 400

Access token is missing in the request.

HTTP status code 401

Access token is not valid (it expired, was revoked, uses wrong encryption keys or is damaged).

HTTP status code 403

User cannot perform this action.
